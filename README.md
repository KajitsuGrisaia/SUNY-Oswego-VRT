<h1>SUNY Oswego Virtual Tour repository</h1>
<p>This application was made for a final project of SUNY Oswego in which a user can interact with a map of the school that shows name of buildings</p>
<p>When the coordinates of the buildings are hovered, an image of the building with appear alongside its information</p>
<p>When the user clicks on the coordinate, they will be redirected to a new website where they can tour through the building</p>
